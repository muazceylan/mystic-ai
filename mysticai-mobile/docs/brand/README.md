- **iOS App Store**: `assets/brand/app-icons/ios/astro-guru-ios-app-store-1024.png`

## Rules

- Do not alter colors or add effects
- Use `small-optimized` or `flat-reduced` for sizes below 64px
- Never place the dark-bg version on a light background
- Keep clear space around the icon (min 25% of G-form width)
- See `astro-guru-mini-brand-guide.md` for full guidelines
